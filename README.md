# Admin Owner Management System

A Spring Boot web application for managing Admins, Owners, and Properties.

## Overview

The **Admin Owner Management System** allows an Admin to manage Owner accounts, and Owners to manage their properties.  
The application follows the **MVC architecture** and uses **Spring Boot + MySQL** for backend processing and data persistence.


## Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML / CSS

## How to Run
1. Create database:
   CREATE DATABASE admin_owner_db;
2. Update MySQL credentials in application.properties
3. Run:
   AdminOwnerAppApplication
4. Open:
   http://localhost:8080/login

## Flow
- Admin logs in
- Admin creates Owner
- Owner logs in
- Owner adds / updates / deletes properties

## Features
- Role based login (Admin / Owner)
- Property CRUD
- Success messages
- Chatbot support
